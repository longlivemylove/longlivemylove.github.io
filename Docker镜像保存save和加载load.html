Docker镜像保存和加载
发表于 2019-10-10| 更新于 2019-11-08|Docker
字数总计: 90|阅读时长: 1 分钟| 阅读量:

查看需要保存的镜像的ID

[root@localhost ~]# docker images
[root@localhost ~]#docker ps

保存镜像

[root@localhost ~]#docker commit 765cfe2feb4d 10.1.1.111:5000/web:1.2.0.6

[root@localhost ~]#docker save 10.1.1.111:5000/web:1.2.0.6 |gzip > ./web.zip

加载镜像

可以在任何装 docker 的地方加载,刚保存的镜像了。

[root@localhost ~]#docker load -i web.zip

[root@localhost ~]#docker run
